SSU

Server-Side Uploader, the data aggregation engine.

Download this project as a .zip file Download this project as a tar.gz file

Collect information from your favorite sites.

SSU was originally designed to power data aggregation for Wesabe, a personal finance website. When Wesabe shut down, SSU was made open source and expanded to include support for more than simply bank websites. You can use it to gather documents, scrape data, and generally mine the web for information you or your business needs.

Get Started

$ git clone https://github.com/wesabe/ssu.git
$ cd ssu
$ ./bootstrap

SSU works out of the box on Mac OS X and any Linux distribution using apt-get.

Data Aggregation

SSU works using per-site scripts – written in CoffeeScript or JavaScript – that know how to do various things such as “download all the transaction history from this user's Chase account”. Each script is named with reverse DNS naming, so a script for banking.example.com would be named com.example.banking and will be located in application/chrome/content/wesabe/fi-scripts/com/example.banking.(js|coffee).

Support or Contact

While there's no paid support available for SSU, you can use the mailing list to ask questions.
